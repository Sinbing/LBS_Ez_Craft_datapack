### 数据包~食用~使用说明  

1. 将数据包放入存档文件夹中。  

   ​	本地存档位于 `\.minecraft\saves\[<存档名>]\datapacks\` ；  
   ​	服务器存档位于 `\world\datapacks\`。  

2. 在游戏中使用 `/reload` 指令。  

3. 在游戏中使用 `/datapack list` 指令，查看已载入数据包。  

   ​	如返回已启用数据包中包含 `[file/Ez_Crafted]` 则安装成功；  
   ​	使用 `/datapack disable` 或 `/datapack enable` 可以分别启用/禁用数据包。
